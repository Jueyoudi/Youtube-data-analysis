# YouTube 中文内容出海：赛道热度、成功频道与驱动因素分析

## 1. 项目目标
本项目面向“中文自媒体出海 YouTube”的选赛道与内容策略问题，基于 YouTube Data API 采集公开视频数据并进行分析，回答：

1) 哪些赛道**最火**（规模热度 vs 公平热度）  
2) 哪些赛道更容易产出**成功频道**（进入全平台 Top10% 的频道）  
3) 各赛道播放量最关键的驱动因素是什么（频道订阅数、时长、发布时间新旧）

---

## 2. 数据与口径

### 2.1 数据采集方式
- 使用 **YouTube Data API v3**
- 采用“少量关键词发现频道 → 获取频道 uploads 播放列表 → 批量拉取频道近期视频”流程扩展样本，降低对高成本 `search.list` 的依赖，并控制配额成本

### 2.2 重新分类口径
- 初始发现频道的类别（seed_category）不等于频道内每条视频的真实内容类别  
- 因此对每条视频基于标题/描述关键词打分进行**重新分类**，得到 `pred_category`（8 大赛道 + `other`）  
- 未命中任何赛道关键词的内容归入 `other`

> 注意：`other` 体量非常大，混合了大量未覆盖/难分类内容（如泛娱乐、儿歌、泛资讯等）。赛道对比时需区分“other”与“明确赛道”。

## 5. 各赛道影响播放量的关键因素（相关 + 回归）

### 5.1 方法说明
- 目标变量：`log(viewCount + 1)`
- 特征变量：`log(subscriberCount + 1)`、`log(duration_sec + 1)`、`age_days`（上线天数）
- 输出：
  - Spearman：稳健相关（单变量视角）
  - beta：标准化多变量回归系数（控制其他因素后的边际影响，绝对值可比较“重要性”）
  - R²：模型解释力

### 5.2 结果表

**解读要点（结论可直接用于策略）**
- **共性：订阅数是所有赛道的第一驱动**（most_important 均为 `log_subs`，且 Spearman(订阅,播放) 最高）。
- **可预测性（R²）**：food（0.657）与 overseas_life（0.598）最高，说明这两类内容播放更“规律”，更受频道体量与供给形态驱动；job_hunting（0.286）与 movie_story（0.337）较低，说明更依赖选题爆点、标题封面、推荐机制波动。
- **时长效应**：
  - 更偏“长内容有利”：movie_story（beta_log_dur=+0.581）、news_current（+0.209）、immigration（+0.129）
  - 更偏“短内容/紧凑有利”：job_hunting（beta_log_dur=-0.429）
- **发布时间新旧（age_days）**：部分赛道 beta_age_days 较大（如 news_current / movie_story / job_hunting），提示存在累积播放或共线性影响；后续可用 `views_per_day` 作为因变量进一步消除“老视频天然累积”偏差。

## 6. 可执行建议（选赛道与内容形态）

- **追热点/快速起量**：news_current（均值日均播放最高），但波动大、需要高频更新与热点捕捉能力  
- **稳定增长 + 爆款能力**：food、podcast_interview（Top10%视频均播放高；且 food 的规律性最强）  
- **高互动/高转化（咨询、课程、社群）**：study_abroad（评论率最高）与 immigration（评论率较高），适合“攻略/避坑/清单/模板”内容承接转化  
- **内容形态建议**：
  - movie_story / news_current / immigration：更适合深度叙事或解读型长内容  
  - job_hunting：更适合短平快、高密度、结论先行的表达方式

---

## 7. 局限性与下一步
- `other` 类别混合度高，建议进一步细分（娱乐/儿歌/泛资讯等）  
- 频道总播放基于“每频道近期 N 条视频”窗口，并非全历史全量  
- 影响因素仅覆盖订阅/时长/新旧，后续可加入标题特征、主题聚类、是否 Shorts、封面风格代理等  
- 建议使用 `views_per_day` 作为目标变量，进一步验证“发布时间新旧”的真实影响
